## Building

Requires [Node.js](https://nodejs.org/).

```bash
npm install
npm run pack
```

This produces two files in the project root:

- `old-rblx-logo-chrome.zip` (Manifest V3)
- `old-rblx-logo-firefox.zip` (Manifest V2)
